    1.2 标识符、常量与变量

  • 标识符命名规则
    • 标识符是由字母、数字、下划线(_)、美元符号($)组成的字符序列。
    • 标识符必须以字母、下划线(_)、美元符号($)开头。不能以数字开头。标识符不能是保留字。
    • 标识符不能为true、false或null等事实上的保留字（参见英文维基网）
    • 标识符可以为任意长度，但编译通常只接受前128字符
    • 例如：$2, area, radius, showMessageDialog是合法的标识符；2A, d+4是非法的标识符
